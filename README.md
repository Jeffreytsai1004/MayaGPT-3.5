## 安装步骤
- 从终端 cd 到运行 mayapy 的 bin 目录mayapy -m pip install mayagpt
```
cd C:\Program Files\Autodesk\Maya2023\bin
```
```
mayapy -m pip install mayagpt
```
- 然后在Maya中运行
```
import mayagpt
mayagpt.show()
```
